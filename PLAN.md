# Redesigned Guides, clear roadmap with how-to steps, PDF export, and AI coach

## What will change

### 1. Guides — open to a full screen with drop-down sections
- Tapping any guide card on the Guides page now pushes to a dedicated full-screen guide view (instead of expanding inline).
- Inside each guide, the categories (e.g. "Week 1: Stabilize", "Health & Wellbeing", "Resume Translation") become collapsible drop-downs.
- A clean hero header at the top of each guide shows the title, subtitle, and an icon in the guide's color.
- First section auto-expands so the user sees content immediately; others tap to open.
- Smooth spring animation when opening/closing sections, with a subtle haptic.

### 2. "You" AI Coach powered by Apple Intelligence
- On supported devices (iPhone 15 Pro and newer with Apple Intelligence enabled), a new **"Where You Are"** card appears at the top of the Plan screen and inside each guide.
- It analyzes the user's current phase, completed tasks, documents secured, benefits started, and days until/since separation — then generates a short personalized message that speaks directly to "You" (e.g. *"You're 4 months out and ahead on admin — now's the time to lock in your resume and start interviewing."*).
- Refreshes on demand with a subtle "refresh" button.
- On devices without Apple Intelligence, falls back gracefully to a smart rule-based summary (no missing UI, no error).
- Runs fully on-device — private, free, offline.

### 3. Education stays as "Education"
- The "Edu" short label is removed everywhere it appears (task category chips, badges, lists). Full word "Education" is used consistently.

### 4. Roadmap — clear, obvious, with step-by-step how-to instructions
- The **Your Roadmap** section on the Plan screen gets a visual upgrade: bigger phase markers, clearer "NOW" indicator, and a more pronounced connecting line between phases.
- Each roadmap phase is tappable and opens a detailed phase screen with:
  - A hero describing what this phase is about.
  - Every task shown as a card with a clear **"How to do this"** section — step-by-step instructions (1, 2, 3…), tips, and any relevant official links.
  - Checkbox to mark complete, plus an expand/collapse for the how-to steps.
- Tasks that already exist keep their progress — no data loss.

### 5. Roadmap PDF export
- New **Export Roadmap to PDF** button at the top of the Plan screen.
- Generates a clean, branded multi-page PDF containing:
  - Cover page with the user's name, current phase, and overall readiness %.
  - All roadmap phases in order, each with their tasks (✓ completed / ☐ open) and short how-to summaries.
  - Planning areas progress (Career, Education, Family, Financial).
  - Documents checklist.
- Opens the standard iOS share sheet so the user can save to Files, email it, print it, or AirDrop.

## What stays the same
- Today, Tools, Learn, Profile tabs — unchanged.
- Paywall, Pro upgrade placement, in-app purchases — unchanged.
- All existing data (tasks, documents, journal, 90-day planner) — preserved.
- No second paywall added.
